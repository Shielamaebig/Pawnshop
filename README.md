# PawnLedger Lending Prototype

A pure frontend pawnshop lending calculator built with Vue 3, TypeScript, and Vite.
Transactions live only in component state and reset to the bundled sample data after a reload.

## Run locally

```bash
npm install
npm run dev
```

## Verify

```bash
npm run build
npm run test:e2e
```

The Playwright smoke tests use a locally installed Google Chrome browser.
